# POKEMMO KNOWLEDGE ASSIST

The general use purpose of this app is to help players from the game POKEMMO to learn and improve at the game. That said, we going to build fun ways like trivias and games to help and improve specific knoledges about pokemon.

## Important Sources

1. Use a public pokemon api to display data about the pokemons

## Game Mechanics

1. Trivia
   All trivias must follow the same following rules:

- Should have a timer of **30 seconds** for the player to respond and if not that question will be considered as incorrect
- The right answers always have to be displayed to the player after his response

## Games

1. POKE TYPE TRIVIA
   This game will be a trivia game that will show a pokemon picture and name and the player have to figure what type the pokemon is. All the types that exists in the leagues kanto, johto, sinoh, unova and howen must be shown as options to select and the right answer could be multiple options.
